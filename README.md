# 1.00115965218: A Stratigraphy of the Electron Magnetic Moment

PIATRA Institute paper reading the experimentally measured value of $|g_e|/2 = 1.00115965218 \ldots$ as historical sediment. Each digit traces to a specific QED calculation or apparatus, from Dirac (1928) through Fan, Myers, Sukra and Gabrielse (2023). The paper formalises the stratigraphic reading, reconstructs the QED perturbation series at orders one through five, and quantifies the fine-structure-constant tension between the 2018 Berkeley caesium and 2020 Paris rubidium measurements that now limits the Standard Model comparison.

- `paper/PAPER.md` — the paper. Build with `bash paper/scripts/build-paper.sh` (requires pandoc + xelatex).
- `simulation/` — reproduces every numeric claim. Run with `cd simulation && uv run run_all.py`.
- `chat.md` — the original draft from which the paper was developed.
